## Writing style

- No em dashes (—) anywhere: site copy, code comments, commits, or docs. Use commas, colons, periods, or parentheses instead. Same for en dashes (–) in ranges; use a plain hyphen.
- Smallest font size on the site is 16px. Smallest font weight is medium (500); never use light or regular.
- Never use a monospace font (no `font-mono`). Use the Switzer sans for all text, including small labels.

## Kontinuo handoffs

Before starting work, use the Kontinuo MCP to read the current handoff for this project.

Before stopping, switching tasks, or handing work to another agent, use the Kontinuo MCP to write a checkpoint with:
- what changed
- what was verified
- what is not done
- the exact next action

If you are unsure about the checkpoint format, call the Kontinuo schema MCP tool first.
