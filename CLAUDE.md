## Kontinuo handoffs

Before starting work, use the Kontinuo MCP to read the current handoff for this project.

Before stopping, switching tasks, or handing work to another agent, use the Kontinuo MCP to write a checkpoint with:
- what changed
- what was verified
- what is not done
- the exact next action

If you are unsure about the checkpoint format, call the Kontinuo schema MCP tool first.
